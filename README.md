# Driver Manager Module

A modern web application for managing drivers and fleets, built with **.NET 8**, **ASP.NET Core Razor Pages**, and **Tailwind CSS**.

> **Note:** This repository is a public showcase containing screenshots and documentation only. The application source code is private and is not included.

---

## Overview

The **Driver Manager Module** provides a complete solution for managing drivers, fleet owners, and fleet relationships within a transportation or logistics platform.

The module allows organizations to manage different driver profiles, including fleet owners, fleet drivers, and independent drivers, through a simple and modern user interface.

---

## Features

### Driver Management

* View and manage the complete list of drivers.
* Add new drivers and define their type:

  * **Fleet Owner**
  * **Fleet Driver**
  * **Independent Driver**
* Create independent drivers without any fleet assignment.
* Add drivers to a fleet through the **Fleet Owner workspace**.
* Modify and manage Fleet Owner information.
* Update independent drivers who are not attached to any fleet.
* Update fleet drivers assigned to a fleet.
* Manage driver and fleet relationships.
